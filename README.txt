 זהו הניסיון שלי למימוש המשימה של אתר משאבי אנוש.
 מהלך העבודה: בשל בעיות בdebugger של VSCode עם מחשב שמקודד בעברית הרבה מזמן העבודה היה בניסיון לפתור בעיה זאת, 
לבסוף בשל אילוצי זמן החלטתי לוותר עליו ועבדתי עם הרצות דרך הcommand line. 
לצערי לא הגעתי במהלך הזמן שהיה לי לתוצאות משמעותיות. זהו הניסיון הראשון שלי בעבודה עם nodejs ועם angular, ולכן התקשיתי יותר משציפיתי.
הגרסה הסופית אליה הגעתי היא עמוד Login פשוט בעל שדות שם משתמש וסיסמה וכפתור כניסה,
ושרת אשר בהינתן הודעת HTTP מסוג POST בעל שדות שם משתמש וסיסמה, בודק האם הם מתאימים למשתמש אשר נמצא בקובץ json ממנו הוא שולף
את המידע, ומחזיר הודעת HTTP מתאימה.
עם זאת, לא הספקתי להגיע לגרסת לקוח אשר מבצעת תקשורת HTTP בצורה מספקת, ולכן אין דרך לבדוק את פונקציונליות השרת באמצעות הלקוח, והם 
אינם מתקשרים אחד עם השני, משום כך, הלקוח אינו בודק כרגע האם השם משתמש והסיסמה שניתנו לו תקינים.
בשל המחסור בצד HTTP בלקוח, השרת נבדק כרגע באמצעות קובץ rest וקליינט מתאים שהותקן בVSC.
אמשיך לעבוד על תרגיל זה בניסיון לסיים אותו ולהתנסות עם טכנולוגיות אלו. תודה על ההזדמנות, גיא.
